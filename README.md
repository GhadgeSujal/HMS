# Hospital Management System

![Angular](https://img.shields.io/badge/Angular-16-red)
![Java](https://img.shields.io/badge/Java-SpringBoot-brightgreen)
![MySQL](https://img.shields.io/badge/MySQL-database-blue)
![Bootstrap](https://img.shields.io/badge/Bootstrap-5-purple)

A web-based Hospital Management System (HMS) built with Angular 16 and Java Spring Boot, designed to manage patients, doctors, appointments, and prescriptions efficiently. The system provides separate login portals for Admins, Doctors, and Patients, streamlining hospital operations.

---

## Features
- Role-based access control
- Login system for Admin and Doctor
- Admin can manage patients, appointments, and view patient details
- Doctors can view appointments, update prescriptions, and manage medicines
- Responsive UI using Bootstrap 5

---

## Technologies Used
- **Frontend:** Angular 16  
- **Backend:** Java Spring Boot, Hibernate, JPA  
- **Database:** MySQL  
- **UI Framework:** Bootstrap 5  

---

## API Endpoints
- `GET /patients` → List all patients
- `POST /appointments` → Create new appointment
- `PUT /medicine/{id}` → Update medicine

---

## Future Features

---

## Technologies Overview / Why Used

- **Angular 16 (Frontend):**  
  Component-based SPA framework for building responsive and interactive dashboards.

- **Java Spring Boot (Backend):**  
  Simplifies building scalable REST APIs with easy database integration.

- **Hibernate & JPA:**  
  Efficient ORM for smooth database management.

- **MySQL (Database):**  
  Reliable relational database for patient, doctor, and appointment records.

- **Bootstrap 5 (UI Framework):**  
  Creates a clean, responsive, and mobile-friendly user interface.
